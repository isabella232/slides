Easy to Use

// Listen for events
PUBNUB.subscribe({
  channel: "my_channel",
  message: function (message) {
    console.log(message);
  }
});

// Send Events
PUBNUB.publish({
  channel: "my_channel",
  message: "Hello World!"
});
And now how is this built on our system? It is simple: pub and sub. We do the rest!

PubNub is a Globally Distributed Real-Time Network

The core building-blocks needed to build and scale real-time apps, dramatically reducing the cost, complexity, and time to deliver large-scale real-time experiences.

We at PubNub want to solve this issue for developers by building the core building blocks for real time applications.
